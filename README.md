# Movie app

It is a Octet company case to test the qualifications of the candidates.

## Start dev server

> npm run start

-   starts on <a href="http://localhost:8080/" target="__blank">http://localhost:8080/</a>

## Build for production

> npm run build

-   builds into '/dist' folder

## Database server

> npm run db

-   starts at port 3001
